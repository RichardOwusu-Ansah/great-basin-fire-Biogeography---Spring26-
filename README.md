# Biogeography Research Workflow

This project is organized to support research on post-fire cheatgrass occurrence in the Great Basin region. The main goal is to identify, compare, and prepare datasets for analyzing whether cheatgrass occurrence increases faster and more strongly in burned areas than in comparable unburned areas. The workflow also leaves room to examine possible biodiversity impacts associated with wildfire and cheatgrass spread. The structure is set up to keep dataset handling documented, reproducible, and easy to follow.
